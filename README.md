## Versão em Português

# VT — Virtual Tag

Durante minhas atividades como back office, o endereçador dos Correios sempre foi uma ferramenta essencial.  
Com a aproximação da sua descontinuação, decidi criar meu próprio sistema de geração de etiquetas para manter meu fluxo de trabalho eficiente.

---

## 🚀 Tecnologias Utilizadas
- **HTML**
- **TailwindCSS**
- **JavaScript**
- **jsPDF**
- **JsBarcode**
- **API ViaCEP**

A API ViaCEP permite que o usuário insira apenas o CEP e tenha grande parte do endereço preenchida automaticamente.

---

## 🧩 Desafios Encontrados
Eu programo majoritariamente em **Python** e atualmente estou estudando **React** para aprimorar minhas habilidades de front-end.  
Por isso, trabalhar com jsPDF foi tanto uma solução quanto um desafio, a biblioteca é poderosa, mas exigiu bastante pesquisa e tentativa e erro.

Com apoio de documentação e experimentações, consegui implementar tudo que precisava.  
O desenvolvimento completo levou cerca de **seis horas**:  
- ~2h para HTML, Tailwind e JavaScript puro  
- alguns minutos para JsBarcode  
- o restante para configurar e ajustar o jsPDF

---

## 📦 O que o sistema faz?
O VT — Virtual Tag gera etiquetas de envio para encomendas em geral, otimizando o tempo de preparação e permitindo salvar endereços usados com frequência.

---

## 📌 Status
O sistema está **pronto para uso** e totalmente funcional.

## English Version

# VT - Virtual Tag

During all my time working as a back-office, the Correios address labeling system was an essential tool.
Since it is about to be discontinued, I decided to create my own system for this necessity, thus keeping my workflow efficient.

---

## 🚀 Technologies Used
- **HTML**
- **TailwindCSS**
- **JavaScript**
- **jsPDF**
- **JsBarcode**
- **API ViaCEP**

The ViaCEP API allows the user to enter only the postal code and automatically fill in most of the address fields.

---

## 🧩 Challenges Found
I mostly program in Python, and at the moment I'm learning React to improve my front-end skills. Because of that, working with jsPDF was both a solution and a challenge. The library is powerful, but required a lot of research and trial-and-error.

With documentation, experimentation, and AI assistance, I was able to implement everything I needed. The complete development took approximately six hours: about two hours for the HTML, Tailwind, and JavaScript, a few minutes for JsBarcode, and the rest for configuring and fine-tuning jsPDF.

---

## 📦 What does the system do?

VT — Virtual Tag generates shipping labels for general parcels, optimizing preparation time and allowing you to save frequently used addresses.

---

## 📌 Status
The system is **ready for use** and fully functional.
